# 🌱 Site Institucional — Projeto Web

Projeto de **site institucional responsivo**, desenvolvido com foco em **boas práticas de front-end**, organização de código e **integração com PHP** para simular um formulário de contato real.

---

## 🚀 Tecnologias
- HTML5  
- CSS3  
- JavaScript (ES6)  
- PHP  

---

## ✨ Funcionalidades
- Header fixo com navegação suave  
- Banner com carrossel  
- Seções institucionais e grid de produtos  
- Animações ao scroll  
- Formulário de contato com validação em PHP  

---

## ⚠️ Observações
- O **GitHub Pages não executa PHP**, portanto o formulário não funciona nessa plataforma.  
- Para rodar localmente no **macOS**, é necessário um ambiente com PHP e **configuração de SMTP**, pois o envio de e-mails via `mail()` não funciona por padrão.

---
